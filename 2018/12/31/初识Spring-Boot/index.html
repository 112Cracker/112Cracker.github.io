Spring Boot 介绍

Sprint Boot是Spring开发的众多项目之一。通过Spring Boot，开发者可以使用Java语言便捷高效地进行服务器端的Web开发。尽管Spring Boot不是唯一的基于Java的框架，开发者利用Spring Boot可以简洁高效地开发独立的，可投入生产的，基于Spring生态的应用，正如官方文档中所说，”Applications that you can ‘just run’”。Spring Boot在适用于编写小型应用解决实际问题的同时，其真正的强项在于使用Spring Cloud构建大型的，分布式的应用程序，在这种情境下，通常有多个应用程序进行协同工作。

Spring Boot 2.0 的新特性：

  • 编程语言：Java 8+, Kotlin
  • 底层框架：Spring Framework 5.0.x
  • 全新特性： Web Flux

为什么选择Web Flux?

  • 函数编程：Java 8 Lamdba
  • 响应式编程：Reative Streams
  • 异步编程： Servlet 3.1 或 Async NIO
